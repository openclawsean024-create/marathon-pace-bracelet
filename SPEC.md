## 1. 專案目標（Project Goals）

### 核心使命
打造一款專為臺灣馬拉松跑者設計的「配速視覺化工具」，讓跑者在賽事前就能清楚掌握全程配速策略，並在賽中無需掏手機即可參考。

### 功能需求層級

| 層級 | 目標 | 說明 |
|-----|------|------|
| **P0（必備）** | 配速計算核心 | 輸入目標時間 → 自動生成每公里配速表 |
| **P0（必備）** | 分段視覺化 | 公里分段時間一目了然 |
| **P1（重要）** | 體力調整機制 | 根據上下坡、體力狀況動態調整 |
| **P1（重要）** | 離線可用 | 賽中無網路也能查詢 |
| **P2（加分）** | 賽事路線圖 | 視覺化呈現賽道特性 |
| **P2（加分）** | 配速策略建議 | 內建專業配速策略模板 |

### 主要用戶

| 族群 | 輪廓 | 需求 |

## 5. 路線圖設計方向（Route Map Design Direction）

> **⚠️ 重要宣告：不使用 GPS 高度圖**

### 5.1 為什麼不做真實 GPS 高度圖？

| 問題 | 說明 |
|------|------|
| **精確度不足** | GPS 高度誤差不穩定（±10-30m），無法準確呈現真實起伏 |
| **實用價值低** | 跑者需要的是「策略建議」，不是精確高度數值 |
| **開發成本高** | 需要處理大量軌跡資料、校正、相位對齊 |
| **競品已做爛** | 現有 App 的高度圖多數是示意，參考性低 |

### 5.2 替代方案：靜態視覺化路線圖

採用「**路線海報 / 路線簡報卡**」風格，以靜態圖片呈現賽道特性：

#### 設計原則

```
┌─────────────────────────────────────────────────────┐
│                  [賽事名稱] 路線圖                    │
├─────────────────────────────────────────────────────┤
│                                                     │
│     ▲ 上坡預警標記        ─── 主要平路段             │
│     ▼ 下坡提示            ····· 起伏段               │
│                                                     │
│     [公里數標註]           [特色地標/景點]           │
│                                                     │
│     0K ─────── 10K ─────── 21K ─────── 42K          │
│     │                        │                       │
│     起點                    半程                     │
│                                                     │
│     ████████████  平路  ████████████                │
│     ▓▓▓▓▓▓▓▓▓▓▓▓  起伏  ▓▓▓▓▓▓▓▓▓▓▓▓              │
│                                                     │
└─────────────────────────────────────────────────────┘
```

#### 呈現元素

| 元素 | 說明 | 範例 |
|------|------|------|
| **路線剖面示意** | 以簡化線條呈現高低起伏趨勢（非精確高度） | 波浪線條 |
| **公里數標註** | 每5K標註，便於對照配速卡 | 5K, 10K, 15K... |
| **上下坡預警** | 以 ▲/▼ 符號標示明顯上坡/下坡區段 | 30K-32K ▲▲▲ |
| **特色景點** | 標示有趣的地標/景點（拍照點/补给站） | 故宮、淡金公路 |
| **難易度指標** | 以顏色區塊表示該段難度 | 綠/黃/紅 |

#### 靜態海報的優勢

1. **離線友好**：一張圖片，無需網路
2. **製作快速**：設計一次，長期使用
3. **可客製化**：可疊加跑者自己的配速數據
4. **美觀大方**：如同官方路線海報，有收藏價值

### 5.3 實作方式（給 Alan 的方向）

```
技術建議：
1. 路線圖以 SVG 或 PNG 靜態圖片為主
2. 公里數標註可使用絕對定位疊加圖層
3. 上下坡資訊以 JSON 資料結構儲存（易於更新）
4. 響應式設計：手機豎向 / 橫向皆可瀏覽
5. 圖片格式建議：SVG（可縮放）+ WebP（壓縮）
```

## 6. 完整配速策略章節（Pacing Strategy）

> **核心原則：配速不是一成不變的數字，而是動態的體力管理策略**

### 6.1 三大主流配速策略

#### 策略 A：負分段配速法（Negative Split）⭐ 推薦

**原理**：後半程比前半程快，是最科學、最被廣泛建議的策略。

```
時間軸：  0K ──────────── 21K ──────────── 42K
          │    前半馬      │     後半馬      │
          │                │                │
配速：    慢 5-10秒/K  →  基準配速  →  快 5-10秒/K
累積差：  +30秒~1分      0           -30秒~-1分
```

**適用人選**：
- 全馬目標 SUB-4 的跑者
- 有經驗但希望穩定完賽
- 不想冒險的保守型跑者

**具體執行表（以SUB-4為例）**

| 段落 | 目標配速 | 每K時間 | 累計時間 | 策略重點 |
|------|---------|---------|---------|---------|
| 1-5K | 5:40/K | 5:40 | 28:20 | 熱身期，刻意慢跑，避免興奮過度 |
| 6-15K | 5:30/K | 5:30 | 1:22:30 | 進入配速，保持鼻子呼吸 |
| 16-25K | 5:25/K | 5:25 | 2:15:00 | 舒適區間，維持節奏 |
| 26-32K | 5:20/K | 5:20 | 2:51:40 | **撞牆期前**，加速儲備體力 |
| 33-40K | 5:15/K | 5:15 | 3:27:40 | 負分段發力，超越自己 |
| 41-42.195K | 5:10/K | 0:20 | 3:59:xx | 最後衝刺，享受掌聲 |

#### 策略 B：均速配速法（Even Pace）

**原理**：全程幾乎相同的配速，最簡單但難度最高。

**適用人選**：
- 高手級（SUB-3）
- 有精準配速能力的跑者
- 天氣條件極佳時

**注意事項**：
- 必須非常了解自己的能力
- 天氣變化會大幅影響執行
- 建議搭配心率監控

#### 策略 C：正向分段（Positive Split）

**原理**：前半快、後半慢，**不建議作為主要策略**。

| 狀況 | 說明 |
|------|------|
| 原因 | 前半馬體力充沛容易跑快，後半體力下降導致崩潰 |
| 數據 | 90%的業餘跑者會遇到「撞牆期」 |
| 建議 | 除非有特殊策略（如前半刻意領先），否則避免 |

### 6.2 體力分配黃金比例

#### 基础体力 model

```
能量系統分配：
┌──────────────────────────────────────────────────────┐
│  磷酸系統（ATP-PCr）  │  3-5分鐘  │  瞬發力/衝刺用    │
│  醣解系統（Glycolysis）│  30-90分  │  馬拉松主力系統   │
│  氧化系統（Oxidative） │  2小時+   │  脂肪供能/長距離  │
└──────────────────────────────────────────────────────┘
```

#### 馬拉松體力消耗曲線

```
消耗程度 │
    高   │                    ╭── 撞牆期
         │               ╭──╯
         │          ╭──╯
    低   │     ╭──╯
         │╭──╯
         └─────────────────────────────→
           0K    10K    21K    30K    40K   42K
```

#### 各階段體力管理

| 階段 | 里程 | 體力狀態 | 策略 |
|------|------|---------|------|
| **黃金前期** | 0-10K | 90-100% | 保守起步，為後段儲能 |
| **穩定期** | 10-25K | 75-85% | 進入巡航配速，保持節奏 |
| **關鍵期** | 25-32K | 60-70% | **撞牆期**，意志力考驗 |
| **突破期** | 32-38K | 55-65% | 負分段策略開始加速 |
| **衝刺期** | 38-42K | 50-60% | 剩餘體力全部釋放 |

### 6.3 撞牆期（The Wall）應對策略

#### 什麼是撞牆期？
- 通常發生在 30-35K
- 肝醣耗盡、血糖低、肌肉疲勞同時發生
- 配速突然崩跌（每公里慢30秒到1分鐘）

#### 預防策略

| 策略 | 說明 | 執行時機 |
|------|------|---------|
| **前半慢5-10秒** | 保留體力 | 0-10K |
| **碳水化合物補充** | 賽前72小時高碳水飲食 | 賽前3天 |
| **赛中補水** | 每5K補充水分，逢站必喝 | 比賽中 |
| **能量膠策略** | 每30分鐘攝取一份碳水 | 比賽中 |
| **配速預冷** | 負分段提前加速 | 30K前 |

#### 當下應對

```
撞牆當下怎麼辦？

Step 1：不要停。降低配速10-15秒，不要走路
Step 2：補充能量膠或運動飲料
Step 3：深呼吸，鼻子吸氣嘴巴吐氣
Step 4：專注在「一步、一步」，不要想整體距離
Step 5：想像終點的畫面，保持動機
```

### 6.5 天氣因子的配速調整

| 天氣條件 | 調整幅度 | 說明 |
|---------|---------|------|
| **炎熱（>25°C）** | 每K慢10-15秒 | 提早起跑、降低預期 |
| **悶熱（>30°C+高濕）** | 每K慢20-30秒 | 保守再保守 |
| **下雨** | 每K慢5-10秒 | 注意路面濕滑 |
| **強風** | 順風快/逆風慢 5-10秒 | 隨時調整 |
| **最佳條件（15-18°C）** | 標準配速 | 把握機會 |
